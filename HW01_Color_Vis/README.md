# 图片颜色聚类可视化分析系统说明文档

## 1. 项目简介

本系统是一个基于 **Flask** 后端与 **ECharts** 前端的交互式 Web 应用。

它利用机器学习中的 **K-means 聚类算法** 对上传图片的像素颜色进行分析，提取出图片的聚类中心，并以可视化图表的形式展示颜色的分布比例及和谐度评价。 

本系统旨在帮助设计师和艺术学习者通过数据量化的方式理解图片的色彩构成。

------

## 2. 核心功能

### 2.1 基本功能

- **颜色提取**：使用 `scikit-learn` 的 K-means 算法对像素进行聚类。
- **多维度展示**：同时展示聚类中心的**颜色值**以及该颜色在原图中的**像素占比**。
- **专业可视化**：集成 `ECharts` 库，支持**饼图**和**柱状图**的动态渲染。

### 2.2 进阶交互功能

- **1.动态参数$K$调节**：用户可交互式选择聚类中心数量 $K$支持$K$值$2-15$。
- **2.多图切换**：支持从图片库中选择不同的测试图片进行实时分析。
- **3.可视化样式切换**：支持在饼图与**柱状图**之间无缝切换。
- **4.色彩空间转换**：支持 **RGB**与 **LAB**色彩空间的聚类切换，提高颜色提取的准确性。
- **5.AI 智能评估**：调用大语言模型接口，对提取出的配色方案进行“和谐度”评估并给出专业设计建议，如果无法使用请自己更改api（测试可行，可能过期）。

## 4. 安装与运行指南

### 4.1 环境安装

在终端执行以下命令安装必要的 Python 依赖库：

Bash

```
pip install Flask scikit-learn numpy pillow scikit-image requests -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 4.2 运行项目

1. 将你的测试图片放入 `static/images/` 文件夹下。

2. 在项目根目录下打开终端，输入：

   Bash

   ```
   python app.py
   ```

3. 当终端显示 `* Running on http://127.0.0.1:8000` 时，在浏览器打开该地址即可。
